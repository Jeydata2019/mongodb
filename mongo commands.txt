
These are basic commands to interact with Mongo

Database Commands:

1. Show Databases:

show databasesuse

2. Switch Database:

use <database_name>


3. Show Collections:

show collections

4. Create Database:

use <new_database_name>

use database1

Collection Commands:

1.Insert Document:

db.<collection_name>.insert({ key: "value" })

db.collection1.insert({ user_id: 123 })
db.collection1.insert({ user_id: 456 })
db.collection1.insert({ user_id: 789 })

2.Find Documents:

db.<collection_name>.find()

db.collection1.find()

3.Query with Filter:

db.<collection_name>.find({ key: "value" })

db.collection1.find({ user_id: "456" })

4.Update Document:

db.<collection_name>.update({ key: "value" }, { $set: { new_key: "new_value" } })

db.collection1.update({ user_id: 456 }, { $set: { user_id: "010" } })

5.Delete Document:

db.<collection_name>.remove({ key: "value" })

db.collection1.remove({ user_id: "010" })

6.Drop Collection:

db.<collection_name>.drop()

db.collection1.drop()

show collections


7.Indexing:

db.<collection_name>.createIndex({ key: 1 })

General Commands:

1.Show Help:
help

2.Exit MongoDB Shell:
exit


For detailed information and a full list of commands, you can refer to the documentation.

https://docs.mongodb.com/manual/mongo/#mongo-shell
